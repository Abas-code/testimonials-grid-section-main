### What I learned

I learnt how to use CSS Grid.

```css
main {
  display: grid;
  grid-gap: 24px;
}
.daniel { grid-column: 1 / span 3; grid-row: 1; }
.patrick { grid-column: 2 / span 3; grid-row: 2; }
.kira { grid-column: 5; grid-row: 1 / span 2; }
```
